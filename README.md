# FLM GameDay Atlanta

A WordPress plugin for importing and managing sports content from Field Level Media API with AI enhancement, social posting, and analytics.

## Features

- **Content Import**: Automatically import articles from Field Level Media for Atlanta sports teams (Braves, Hawks, Falcons, UGA, Georgia Tech)
- **AI Enhancement**: Generate summaries, headlines, and social posts using Claude AI
- **Social Integration**: Auto-post to Twitter/X and Facebook
- **Analytics**: Track pageviews, engagement, and content performance
- **OAuth Integration**: One-click connection to Google Analytics, Search Console, Twitter, and Facebook
- **ESP Integration**: SendGrid and Aigeon email service provider support
- **Scheduled Imports**: Automatic content syncing via WordPress cron

## Requirements

- WordPress 5.8+
- PHP 7.4+
- Field Level Media API key

## Installation

1. Download the latest release from the [Releases page](https://github.com/MainlineMediaGroup/flm-gameday-atlanta/releases)
2. In WordPress admin, go to **Plugins → Add New → Upload Plugin**
3. Upload the zip file and activate
4. Go to **Settings → FLM Importer** to configure

## Configuration

1. Enter your FLM API key
2. Select which teams to import content for
3. Configure post settings (status, author, category)
4. Optionally connect AI and social integrations

## Automatic Updates

This plugin supports automatic updates directly from GitHub. When a new version is released, you'll see an update notification in your WordPress admin just like any other plugin.
